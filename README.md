# PowerHouse Gym 🏋️‍♂️

### "Strength isn’t given. It’s forged."

A premium, state-of-the-art Gym Management System and PWA (Progressive Web App) designed for modern fitness facilities.

---

## 🌟 Key Features

- **Progressive Web App (PWA)**: Installable on iOS and Android with a dedicated "Add to Home Screen" experience.
- **Multi-Branch Management**: Seamless coordination across multiple locations (Indira Chowk & Pathik Chowk).
- **Interactive Dashboards**: Real-time analytics for Owners, Trainers, and Clients.
- **Smart Attendance**: Automated tracking and management for gym sessions.
- **Membership Management**: Tiered plans (Pro, Elite, Sports & Fitness) with easy signup flows.
- **Trainer Portfolios**: Dedicated spaces for trainers to manage their schedules and clients.

## 🛠 Tech Stack

- **Frontend**: [Next.js](https://nextjs.org/) (App Router), [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/), [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Backend**: [Supabase](https://supabase.com/) (Auth, Database, Edge Functions)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Visuals**: Three.js & Custom 3D Hero Scenes

## 📱 PWA Support

The application is fully optimized as a PWA, featuring:
- **Offline Capabilities**: Access basic information even without an internet connection.
- **One-Tap Install**: Quick installation from the browser directly to the phone's home screen.
- **Fast Loading**: Optimized assets and service worker caching for a smooth experience.
